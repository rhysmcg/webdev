Brisbane Couture Collective: About the Business

By Isabella Singh

Our Mission: Showcasing Brisbane’s top fashion designers, the Brisbane Couture Collective by Isabella Singh focuses on promoting local fashion talent. Our galleries highlight designer collections and upcoming events, connecting fashion lovers with Brisbane's dynamic scene.

Client’s Needs: Isabella requested an interactive map for event locations, multiple designer galleries, and event information for easy access and navigation. This portfolio meets these requirements and aims to provide an immersive experience for visitors.

Website Development Process

Development Overview

Hardware & Software: We used a laptop with sufficient storage and Visual Studio Code for development, applying HTML, CSS, and JavaScript to create an intuitive, visually appealing website. Python was considered for advanced functionalities.

Challenges: Navigating client expectations required detailed communication, and testing across devices helped us refine the responsive design for consistency.

Website Feedback and Recommendations

Evaluation and Future Enhancements

Client Feedback: Isabella appreciated the user-friendly layout and extensive photo galleries. However, she suggested adding a live event feed to engage the audience further.

Future Improvements: We recommend implementing a live event tracker, optimizing loading speed, and potentially adding a shopping cart for direct purchases from designers.
